# Frontend Analyzer

Analyze frontend structure and generate a frontend skill.

## Instructions

Explore the codebase to understand:
- Framework (React, Vue, Svelte, etc.)
- Component organization
- State management
- Routing approach
- Styling solution
- Build tooling

Look for:
- Component files
- Page/route components
- State management setup
- Style files/configuration
- Build configuration

Create the folder `.claude/skills/frontend/` and generate `SKILL.md` inside it:

```markdown
---
name: frontend
description: Explains this project's frontend architecture, components, and UI patterns
---

# Frontend Guide

## Framework
[Framework and key libraries]

## Component Structure
[How components are organized]

## State Management
[How state is managed]

## Routing
[How routing works]

## Styling
[Styling approach and conventions]

## Creating Components
[How to create new components following project patterns]
```

Return a summary of the frontend findings.
